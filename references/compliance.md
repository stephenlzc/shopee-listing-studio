# 蝦皮合規規則

台灣蝦皮平台的禁用詞規範與替換策略。

## 規則來源

- 台灣食品藥物管理署 (TFDA) 化粧品標示法規
- 蝦皮賣場內容審查標準
- 公平交易法廣告不實規範

## 禁用詞分類與替換

### 功效承諾詞 (禁止)

| 禁用詞 | 替換 | 原因 |
|--------|------|------|
| 美白 | 透亮感 | 涉及藥用美白宣稱 |
| 祛斑 / 淡斑 / 除斑 | 日常保養 | 涉及療效 |
| 祛痘 / 去痘 | 清爽保養 | 涉及療效 |
| 消炎 | 日常呵護 | 涉及醫療 |
| 抗皺 / 抗老 / 除皺 | 熟齡保養感 | 涉及療效 |
| 緊緻 / 拉提 | 彈潤感 | 涉及生理改變 |
| 去角質 | 溫和清潔 | 涉及療效暗示 |
| 縮毛孔 | 細緻肌膚 | 涉及生理改變 |
| 控油 | 清爽調理 | 中性化表述 |
| 止汗 | 體感清爽 | 涉及生理改變 |

### 醫療相關 (禁止)

| 禁用詞 | 替換 |
|--------|------|
| 治療 / 藥用 / 療效 / 治癒 / 痊癒 | 日常使用 / 日常保養 / 日常呵護 |
| 處方 | 配方 |
| 藥品 / 藥膏 | 產品 / 保養品 |

### 誇大宣稱 (禁止)

| 禁用詞 | 替換 |
|--------|------|
| 100% 有效 / 立即見效 / 一夜變白 | 日常保養 |
| 瞬間 | 快速 |
| 全網爆款 / 秒殺 | 熱門推薦 |
| 神器 / 黑科技 / 神級 | 好物 / 創新配方 / 優質 |
| 零瑕疵 / 完美 / 永久 | 日常保養 / 理想 / 長效 |

### 敏感族群 (禁止)

| 禁用詞 | 替換 |
|--------|------|
| 敏感肌專用 / 孕婦可用 / 嬰幼兒可用 / 醫美術後 | 初次使用建議先做局部測試 |
| 過敏性肌膚 | 肌膚類型因人而異，建議先做局部測試 |

### 絕對化用語 (禁止)

最高/最低/第一/唯一/最好/最佳/最強/全球 → 優質/合理/熱門/推薦/理想/有感/市售

### 比較性宣稱 (禁止)

比專櫃好/超越醫美/勝過雷射/取代手術 → 日常好選擇/日常保養

### 保證性用語 (禁止)

保證有效/保證見效/無效退費/保證滿意 → 有感保養/安心選購

## 自動過濾

系統在 `utils/compliance.ts` 中實現了 50+ 組自動替換規則。Listing 生成時會自動過濾所有文案（SEO 標題、產品描述、SKU 內容）。
