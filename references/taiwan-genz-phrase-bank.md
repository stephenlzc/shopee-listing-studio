# Taiwan Gen Z 文案短語庫

用於蝦皮商品文案的台灣 Gen Z 語氣參考。

## 語氣定位

- 親切大姐姐陪買家判斷，自然、貼近、實用
- 避免生硬用詞：這款產品、本產品、該產品、此商品
- 像 Dcard / IG / 小紅書的分享語氣

## 開場句型

- 如果你也喜歡...那這罐真的可以收
- 日常保養想找...的話，我會把它放在...
- 這瓶我真的用掉好幾罐了
- 身為...人，這款真的很可以
- 來分享一個我最近的保養新歡

## 中段句型

- 它的質地是...推開之後...
- 用了大概...之後，覺得...
- 最大的差別是...
- 不會...也不會...
- 真的很適合...

## 收尾 CTA

- 有在觀望的可以直接下手了
- 趁免運帶一罐試試看
- 連結放主頁了，自己去看
- CP 值真的高，推推
- 缺貨的話先按補貨通知

## 常用形容詞 (Gen Z)

| 傳統 | Gen Z |
|------|-------|
| 非常好用 | 超好用 / 很可以 |
| 效果很好 | 有感 / 真的有差 |
| 推薦 | 推 / 推推 / 大推 |
| 品質好 | CP 值高 / 很值得 |
| 回購 | 回購率超高 / 用掉好幾罐了 |
| 適合 | 很適合...人 / ...人必收 |

## 常用 emoji 分佈

- 開頭 1 個
- 中間段落 1-2 個  
- 結尾 1 個
- 自然分佈，不要全部放開頭或結尾
- 常見：✨ 💧 🫧 💕 🌿 🧴 💫 🌸

## 禁止

- 迷因雜亂 / 過度搞笑
- emoji 牆 (連續超過 3 個)
- 折扣圖形 / 促銷感過重
- 過度使用「神器」「黑科技」「秒殺」
- 「儀式感」「靈感」等過度抽象詞
